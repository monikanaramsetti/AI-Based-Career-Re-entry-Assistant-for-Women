# Screenshots Directory

This folder contains all project screenshots used in the README.

## Required Screenshot Files

Please add the following screenshot images to this directory:

### 1. landing-page.png
- Screenshot of the landing/home page
- Shows the hero section with "Restart Your Career with Confidence" heading
- Include the "Start My Comeback Plan" button

### 2. career-story-form.png
- Screenshot of the Career Story Form page
- Shows all form fields: Name, Job Role, Years of Experience, Break Duration, Reason, Skills
- Include the submit button

### 3. dashboard.png
- Screenshot of the AI Analysis Dashboard
- Shows readiness score (78%), skill gap analysis
- Include priority skills to learn and action plan phases
- Shows "Welcome back, [User Name]" greeting

### 4. job-opportunities.png
- Screenshot of the Job Opportunities page
- Shows job listings with filters and search
- Display job cards with company info and "View & Apply" buttons

## How to Add Screenshots

1. Run your app: `npm run dev`
2. Take screenshots of each page
3. Save as PNG files with names listed above
4. Place them in this folder

## File Format
- **Format**: PNG (.png)
- **Recommended size**: 1200x800px (adjust as needed)
- **Naming**: Use exact names as listed above (case-sensitive)
